Predicting $I_{n}(t)$ is hard in complex networks. We instead solve for $P(V,t)$

\begin{equation*}
\tau\frac{\partial P}{\partial t} = (\mu(t)-V)\frac{\partial P}{\partial V} + \sqrt{2D}\frac{\partial^{2}P}{\partial V^{2}}
\end{equation*}
